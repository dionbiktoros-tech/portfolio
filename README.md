# 🌸 Beauty Store - Мой проект

## Описание
Это мой проект интернет-магазина косметики, разработанный на Python с использованием Flask.

## Приложения

### 🖥️ Десктопное приложение (EXE)
- Управление магазином косметики
- Работа с товарами, сотрудниками, продажами
- ABC/XYZ анализ
- [Скачать EXE](https://raw.githubusercontent.com/dionbiktoros-tech/portfolio/main/downloads/beauty_store_desktop.exe)

### 📱 Мобильное приложение (APK)
- Просмотр каталога косметики
- Фильтрация по категориям
- Корзина и заказы
- [Скачать APK](https://raw.githubusercontent.com/dionbiktoros-tech/portfolio/main/downloads/beauty_store_mobile.apk)

### 📦 Портативная версия (ZIP)
- Исходный код на Python
- Запуск без установки
- [Скачать ZIP](https://raw.githubusercontent.com/dionbiktoros-tech/portfolio/main/downloads/beauty_store_portable.zip)
