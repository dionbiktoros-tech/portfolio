🌸 Beauty Store Desktop

Настольное приложение для автоматизации магазина косметики с использованием Python и SQLite.

Версия: v1.0

⬇ Скачать EXE ⬇ Скачать БД

Инструкция по установке

  1. Скачайте prilo.exe.
  2. Скачайте beauty_shop.db.
  3. Поместите оба файла в одну папку.
  4. Запустите prilo.exe.
