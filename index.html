💄 Beauty Store

Мобильное приложение для покупки косметики v1.0.1 – v1.0.6
📖 Как скачать и установить:
1. Выберите нужную версию ниже (Android или ПК).
2. Нажмите на ссылку с файлом — скачивание начнётся автоматически.
3. Для Android: откройте скачанный APK-файл на телефоне и разрешите установку из неизвестных источников.
4. Для ПК: запустите BeautyGarden.exe и следуйте инструкциям установщика. База данных (.db) используется приложением автоматически.
⚙️ Требования: Android 6.0+ для APK, Windows 7+ для ПК-версии.
